# Human Copy — the anti-tell system

LLM prose converges on a recognizable statistical median. Readers (and editors — see Wikipedia's "Signs of AI writing") now spot it instantly. This skill is a refusal list + replacement moves. Apply while drafting, then run the ship checks.

## Ban list (replace on sight)

1. **The AI lexicon:** delve, tapestry, testament, landscape (metaphorical), leverage, seamless, elevate, robust, pivotal, crucial, vibrant, intricate, fostering, harness, unlock, empower, supercharge, streamline, boasts, showcase, underscore, meticulous, garner, best-in-class, cutting-edge, world-class, game-changing. → Plain verbs: use, build, run, teach, help.
2. **Copula avoidance:** "serves as," "stands as," "marks," "represents," "is a testament to." → Say **is**. "X is a nonprofit" beats "X stands as a beacon."
3. **Puffery:** "plays a vital role," "pivotal moment," "rich heritage," "nestled in the heart of," "indelible mark." → Cut, or replace with a verifiable fact.
4. **Trailing "-ing" analysis clauses:** "…, highlighting the importance of…" / "…, reflecting a broader commitment to…" → End the sentence. If the conclusion matters, give it its own sentence with evidence.
5. **Hedging:** "may help," "can potentially," "it is important to note." → Take the position.
6. **Section-summary tics:** "In conclusion," "Ultimately," recap paragraphs, "Despite challenges… continues to…" → End on a concrete image, number, or next step.
7. **Vague attribution:** "experts argue," "studies show." → Name the person/org/number/year, or cut.
8. **Elegant variation:** rotating synonyms to avoid repeating a name (the org → the initiative → the movement). → Repeat the name. Humans do.
9. **Title Case Headings + bold-keyword confetti.** → Sentence case; bold only for genuine emphasis.
10. **Inline-header bullet walls** ("• **Label:** description" ×6) as prose substitute. → Prose first; lists only for truly parallel, scannable items.

## Density rules (devices that are human at low frequency, AI at high)

- **Em-dashes:** legitimately human; the tell is 2–3 per paragraph everywhere. Budget: ≤1 em-dash construction per short section; vary with commas, colons, periods.
- **"Not just X, but Y" / "It isn't X. It's Y":** ≤1 per page, placed at the argument's turn, both halves concrete. "Not just education, but implementation" (two real program phases) passes; "not just a website, but a journey" fails.
- **Triads:** ≤1 per page/section, and only when the three items are real and distinct — never adjective-adjective-adjective filler.
- **Provenance test (client work):** if the device appears in the client's own transcripts/interviews, it's voice; if freshly generated, it's suspect.

## Positive program (the strongest humanizers)

1. **Specificity beats everything.** Names, numbers, places, dates in every section. "47 founders," not "many businesses"; "850ms to 120ms," not "improved performance."
2. **Burstiness.** Vary sentence length hard. Three-word sentences. Then one that runs longer because the thought genuinely needs the room. Read aloud to check rhythm.
3. **Concrete verbs, active voice, positions taken.** "We think X," not "X may be considered."
4. **Microcopy in voice** — buttons say what happens ("Read Justin's story," not "Learn More"); 404s, form errors, empty states, footers get written with personality. This is craft AI never bothers with — so doing it reads unmistakably human.
5. **The founder test:** would the person whose name is on this actually say the sentence out loud? If no, rewrite from how they actually talk.
6. One deliberate imperfection of register per page — a plain aside, a "P.S.", a colloquialism — beats uniform polish.

## Ship checks

- Scan for ban-list words and density violations (grep the draft if long).
- Read aloud once; fix anything you stumble on.
- Verify every number/name/claim is real — specificity only humanizes if it's true.
- Headings: sentence case, each earning its place (no "Why choose us?").
- If a brand-voice skill exists for this client (e.g., `ventriq-voice`), it wins on any conflict — this skill guards the floor, the voice skill sets the ceiling.
